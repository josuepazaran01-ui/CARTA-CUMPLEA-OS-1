# Carta de Cumpleaños - Servidor Local

Este proyecto sirve la página `carta_cumpleaños.html` localmente para que puedas abrirla desde tu celular en la misma red Wi-Fi.

## Cómo usar

1. Abre PowerShell en `C:\tmp`.
2. Ejecuta:

   ```powershell
   node server.js
   ```

3. Verás una línea como esta:

   ```text
   Desde el celular usa: http://192.168.1.10:8080
   ```

4. En tu celular, abre el navegador e ingresa esa dirección.

## Requisitos

- Node.js instalado.

## Notas

- Asegúrate de que el celular y la computadora estén conectados a la misma red Wi-Fi.
- Si tu red bloquea conexiones entre dispositivos, puede que necesites permitir el acceso en el cortafuegos de Windows.
